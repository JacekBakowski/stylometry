sādhyā na bhavanti vā tad utpattau  asat bhāvāt svarga tṛpti sukha ādivat  vaidharmyeṇa ubhayatra api bhavetām sukha mardane ānantaryam anuktam ced na sāmarthya avabodhanāt śabda eka deśa bhūtena tena tad pratipāditam kāla antara anupādānāt  karma svābhāvyataḥ  api ca codyamānasya citra ādeḥ ānantaryam viśeṣaṇam atra tāvat asaṃvādaḥ  virodhaḥ ca uttaratra tu svarga yāna viruddhaḥ  hi bhasmībhāvaḥ sadbhāvaḥ  api ca durlabha yadi syāt  vidhi śabdaḥ  atra na eva adarśanataḥ  bhavet  viśeṣaḥ  asya iti bhedena na upāttam syāt idam tatas vidhi śabde bhaviṣyat tvam phalasya parikalpya hi virodha parihāra syāt  vartamāne api na asti asau phalam ca na bhavet evam bhasmībhāvāt  vidhau  api tad sāmarthyena yāni api kalpanā tām niṣedhati prāyas ca evaṃprakāra tvam artha vādeṣu dṛśyate mantreṣu ca iti te sarve pakṣīkāryā prayatna ta evam sati agnihotra ādi vākyeṣu api mṛṣā artha tā veda vākya eka deśa tvāt citra ādi da vacaneṣu iva na ca api atra avinā abhāva upayogi iti sādhitam saṃjñā iti gamaka tvam ced na tad aṅgam iyam bhavet  gamayantī śrutim dṛṣṭvā kalpyate vyavahāra ta na ca eṣā gamayati artham sambandhe anavadhārite tasmāt  gamakatā paścāt  dhūma ādeḥ iva jāyate sa anaṅgam tadvat eva syāt na iyam dhūma ādibhi samā nirūpite avinā abhāve tatra tad kāritā hi asau gamaka tvena pūrvam ca tatra na utpadyate mati iha vācaka tāyā prāṅ na avinā bhāvitām ati yadā eva ca iha sambandham vṛddhebhya pratipādyate tadā eva gamakaḥ  asya ayam iti na anya svarūpa ta kathayanti kvacid tāvat  boddhavyaḥ  asma āt ayam tu iti kvacid  vācaka iti evam vācyaḥ  ayam iti ca ucyate kvacid uccāritāt śabdāt  dṛṣṭvā artha viṣayām kriyām keṣāñcid  tatra boddhṛ tvam anumānāt  pratīyate etena asmād yata śabdāt arthaḥ  ayam avadhārita tena nūnam imau loke siddhau vācaka śaktikau ittham vācaka tā siddhā saṅkīrṇā api tatas param anvaya vyatirekābhyām niṣkṛṣṭerthe avadhārite bahu jāti guṇdravya karma bheda avalambina pratyayān sahasā jātān śrautala akṣaṇika ātmakān na loka kāraṇa abhāvāt nirdhārayitum icchati bala abala ādi siddhi artham vākya jñāḥ tu viviñcate kakṣyā antarita sāmānya viśeṣeṣu hi durbala sāmānya vacana śabdaḥ  jāyate lakṣaṇa abalāt tena avaśyam vivektavya śabdena vācaka gaḥ śabda iti vijñānam anvaya vyatireka jam tasmāt  gamaka taiva ādau  abhidhāyaka tā puna tad nimittā iti sambandha saṃjñā saṃjñitva lakṣaṇa sambandha niyamaḥ  ayam tu yāvinā abhāvita ucyate asādharaṇa tā balāt vijñāna asatti bhāvāt ca cakṣu karaṇam iṣyate tathā eva iha api sambandha jñānam aṅgam prasiddhita gauravāt  karaṇa tvena matam ced  kena vāryate yathā cakṣuḥ iti idam tu
